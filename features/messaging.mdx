Biovity's messaging system enables direct, professional communication between job seekers and employers throughout the application and hiring process.

## Accessing messages

The messaging feature is accessible from multiple locations:

- **Dashboard navigation**: "Messages" item in the main sidebar
- **Notification badge**: Unread message count displayed on the Messages menu item
- **Dashboard home**: Recent messages widget showing latest conversations
- **Application details**: Quick access to message recruiter from your application page

## Conversation list

Each conversation in the list displays:

### Conversation preview

- **Recruiter information**: Name, company, and position (e.g., "María González, HR Manager at BioTech Solutions")
- **Job title**: The position this conversation relates to
- **Last message preview**: Snippet of the most recent message
- **Timestamp**: When the last message was sent (e.g., "14:30", "Yesterday", "Monday")
- **Unread indicator**: Badge showing number of unread messages
- **Online status**: Green dot when recruiter is currently online
- **Application status**: Color-coded badge (Interview Scheduled, Under Review, Offer Extended)

### Status color coding

- **Blue**: Interview Scheduled - requires attention
- **Yellow**: Under Review - waiting for employer decision
- **Green**: Offer Extended - positive outcome
- **Gray**: Initial contact or general inquiry

### Sorting and filtering

Conversations are automatically sorted:

- Unread messages appear first
- Most recent activity at the top
- Archived conversations hidden by default

## Conversation thread

When you select a conversation, the main panel shows:

### Header information

- Recruiter name and title
- Company name
- Job position being discussed
- Online/offline status indicator

### Message thread

The conversation displays all messages in chronological order:

- **Recruiter messages**: Aligned to the left with recruiter's avatar
- **Your messages**: Aligned to the right, visually distinct
- **Timestamps**: Time sent for each message
- **Read receipts**: Indication when messages have been read
- **Date separators**: Headers showing date changes for multi-day conversations

### Message composition

At the bottom of the conversation:

- **Text input field**: Type your message
- **Send button**: Submit message to recruiter
- **Attachment option**: Add documents or files (if applicable)
- **Character limit**: Reasonable limit to encourage focused communication

## Message notifications

Stay informed about new messages:

### Real-time notifications

- **In-app notifications**: Pop-up or toast notification when signed in
- **Badge updates**: Unread count updates automatically
- **Sound alerts**: Optional audio notification for new messages

### Email notifications

- Receive email when you have a new message (if not logged in)
- Configurable frequency (immediate, daily digest, off)
- Includes message preview and link to respond

### Notification badges

Unread message counts appear:

- On the Messages sidebar menu item
- In the conversation list for each unread thread
- In the browser tab title when messages page is not active

## Messaging best practices

<Card title="Communication tips" icon="lightbulb">
  **Professional tone**
  - Use formal language and proper grammar
  - Address recruiters respectfully (e.g., "Estimado/a [Name]")
  - Proofread before sending
  
  **Timely responses**
  - Reply to recruiter messages within 24 hours
  - Acknowledge interview invitations promptly
  - Set expectations if you need time to respond
  
  **Clear communication**
  - Be concise and specific
  - Answer all questions asked
  - Ask clarifying questions when needed
  
  **Professional content**
  - Stay focused on the job opportunity
  - Avoid overly casual language or emojis
  - Keep discussions relevant to the application
</Card>

## Message features

### Read receipts

Messages show read status:

- **Sent**: Message delivered to recruiter
- **Read**: Recruiter has viewed your message
- **Typing indicator**: See when recruiter is composing a response (when online)

### Message history

All conversations are preserved:

- Full message history accessible anytime
- Search within conversations (if implemented)
- Reference previous discussions during interview or offer stages

### Archive conversations

After hiring process concludes:

- Archive conversations to clean up your active list
- Archived messages remain accessible for reference
- Restore archived conversations if needed

## Recent messages widget

Your dashboard home includes a "Recent Messages" card:

- Shows your 3-5 most recent conversations
- Displays recruiter name and last message preview
- Unread indicator for new messages
- Quick link to open full conversation

## Privacy and safety

Biovity messaging protects your privacy:

### Security measures

- Messages only visible to you and the recruiter
- Encrypted transmission of message data
- No sharing of personal contact information until you choose
- Report inappropriate messages to Biovity support

### Professional boundaries

- Messaging is for job-related communication only
- Recruiters must maintain professional conduct
- Report violations or uncomfortable interactions
- Biovity investigates reported issues

### What to share

- **Do share**: Availability, salary expectations (when asked), relevant experience details
- **Don't share initially**: Personal phone number, home address, financial information
- **Share when appropriate**: Contact details after accepting an offer or for interview logistics
