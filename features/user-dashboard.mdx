## Dashboard features

### Notifications system

Stay informed about important updates:

- **Application updates**: When employers review or update your application status
- **Interview scheduling**: Notifications when interviews are scheduled or rescheduled
- **Job recommendations**: Alerts when new matching jobs are posted
- **Message notifications**: Real-time alerts for new recruiter messages

Notification badges appear in the navigation sidebar to highlight items requiring attention.

### Profile progress tracker

The sidebar includes a profile completion card showing:

- Current completion percentage (e.g., 75%)
- Visual progress bar
- Quick link to complete your profile
- Encouragement to improve visibility to employers

A complete profile increases your chances of being discovered by recruiters and receiving relevant job recommendations.

### User account management

Access account options through the user dropdown in the sidebar footer:

- View your profile
- Access account settings
- Log out securely

## Responsive layout

The dashboard adapts to different screen sizes:

- **Desktop**: Full sidebar navigation with all features visible
- **Tablet**: Collapsible sidebar that can be toggled
- **Mobile**: Icon-only sidebar with tooltips for compact navigation

Sidebar state is automatically saved in cookies, so your preference persists across sessions.

## Personalization

The dashboard personalizes content based on your:

- Scientific profession and specialization
- Job search preferences
- Application history
- Saved job criteria
- Interaction patterns

This ensures you see the most relevant opportunities and actionable insights for your career goals.
